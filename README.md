# Git Commit Writer

Public copy of the skill for writing commit messages from staged Git changes.

## Source

This repository is based on the original Agensi source: https://www.agensi.io/.
The architecture is preserved, and the skill is adapted to write the message body in Russian by default.

## Contents

- `SKILL.md` - the actual Codex skill
- `agents/openai.yaml` - UI metadata for Codex

## Install

Copy the `git-commit-writer` skill folder from this repository into your Codex skills directory.
