# Git Commit Writer

## Русский

Публичная копия скилла для написания commit-сообщений по staged-изменениям в Git.

## Источник

Этот репозиторий основан на оригинальном источнике Agensi: https://www.agensi.io/.
Архитектура сохранена, а скилл адаптирован так, чтобы по умолчанию писать текст коммита на русском.

## Содержимое

- `SKILL.md` - сам скилл для Codex
- `agents/openai.yaml` - метаданные интерфейса для Codex

## Установка

Скопируй папку `git-commit-writer` из этого репозитория в каталог скиллов Codex.
